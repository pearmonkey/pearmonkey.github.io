4. Python-Pandas介绍
Sherry | 2020-12-06 | Python |

终于来到了这次学习的重点——Pandas，之前的教程都是为使用Pandas而打下的基础。Pandas是基于Numpy构建的，让以Numpy为中心的应用变得更简单。

安装：使用Pandas之前需要安装，如果在Jupyter中执行import pandas as pd报错，说明还没有安装，请参考这篇进行安装Pandas。

使用：由于Pandas是基于Numpy构建的，所以Numpy里的所有计算函数都可以给Pandas应用，一般情况下，将两个包都导入:

# 以后在其他code里，看到pd就是pandas，np就是numpy
# 其实也可以把package里所有的function都import，from pandas import *，但是这不是一个好习惯
import pandas as pd
import numpy as np

Pandas数据结构

Series

Series是一种类似于一维数组的对象，由一组数据（Numpy数组或者Python数组）和一组与之相关的数据标签（即索引）组成。

a = pd.Series([4, 7, -5, 3])
a

左边是索引，右边是值。默认索引是一个0到N-1（N为数据的长度）的整数型索引。Series有values和index两个属性。

print(a.values)
print(a.index)

可以传递定义好的索引，长度应和数组长度一致：

# 给现有的Series定义index
a.index = ['d', 'b', 'a', 'c']
print(a)

a.index = ['d', 'b', 'a', 'b']
print(a['b'])

# 生成Series时定义index
b = pd.Series([4, 7, -5, 3], index = ['d', 'b', 'a', 'c'])
print(b)
print(b.index)
print(b['a']) # 通过索引获取值

b['a'] = 6 # 通过索引修改值
print(b[['a', 'b', 'c']])

Numpy数组的运算（根据条件语句筛选、矢量计算）都会保留索引和值之间的连接：

print(b)
print(b[b > 5])
print(b * 2)
print(np.exp(b))

还可以通过字典Dict来创建Series，字典的key就是索引，value就是值：

x = {'Jeo': 95, 'Tom': 85, 'Alice': 100}
y = pd.Series(x)
print(y)

在Pandas里，用NaN（np.nan）表示缺失值，可以用Pandas的isnull和notnull来检测缺失数据：

y['Bob'] = np.nan
print(y)
print(pd.isnull(y))
print(y.isnull()) # Series也有同样的function
print(pd.notnull(y))

Series的索引最重要的作用是，在算术运算中会按照索引自动对齐，没有对齐到的索引处用NaN填充：

x = pd.Series({'Jeo': 95, 'Tom': 85, 'Alice': 100})
y = pd.Series({'Jeo': 95, 'Tom': 85, 'Bob': 100})
print(x + y)

DataFrame

DataFrame是一个表格型的数据结构，含有一组有序的列，每列可以是不同的值类型（数值，字符串，布尔值等）。DataFrame既有列索引也有行索引，可以被看由Series组成的字典（共用同一个索引）。可以把DataFrame看作是一个二维结构来保存数据的容器。
构建DataFrame的方法：
传入由几个等长的Python数组或者Numpy数组组成的数组：

data = {'state': ['Ohio', 'Ohio', 'Ohio', 'Nevada', 'Nevada'],
        'year': [2000, 2001, 2002, 2001, 2002],
        'pop': [1.5, 1.7, 3.6, 2.4, 2.9]}
df = pd.DataFrame(data)
print(df)
print(df.columns) # 列名是字典的key
print(df.index) # 被自动添加的索引
print(df.dtypes) # object = str or mixed
print(df.shape)

可以指定列的顺序:

df = pd.DataFrame(data, columns = ['year', 'state', 'pop'])
print(df)
print(df.columns) # 按给定的顺序

# 可以指定行索引
df = pd.DataFrame(data, columns = ['year', 'state', 'pop'],
                        index = ['one', 'two', 'three', 'four', 'five'])
print(df)
print(df.index)

通过列索引和行索引获取数据（.loc()）：

# 获取一列 (类型为Series)
print(df['state'])
print(df[['state', 'year']]) # 获取多列

# 获取一行 (类型为Series)
print(df.loc['three'])
print(type(df.loc['three']))

通过行列的的位置获取数据（.iloc()）：

df.iloc[0:2, 0:2] # 取第一第二行，第一第二列

添加和删除列

可以给DataFrame添加列，长度要和原DataFrame的行数一样，如果添加的是一个Series，会精确对齐DataFrame的行索引（因为双方都有行索引），没有匹配上的被填上空缺值：

df = pd.DataFrame(data, columns = ['year', 'state', 'pop'],
                        index = ['one', 'two', 'three', 'four', 'five'])
print(df)

# 添加列
new_col = pd.Series([-1.2, -1.5, -1.7], index = ['two', 'five', 'four'])
df['new_col'] = new_col
print(df)

df['eastern'] = (df.state == 'Ohio')
print(df)

# 删除列
print(df.columns)
df = df.drop(['new_col', 'year'], axis = 1) # axis = 1 表示按列操作
print(df)

# 删除行
print(df.drop(['one', 'two'], axis = 0)) # axis = 0 表示按行操作
print(df.drop(df.index[df.eastern], axis = 0)) # drop eastern为True的行
print(df.drop(df.index[0:2], axis = 0)) # drop前两行

修改索引

行索引和列索引不能原地修改，可以用.rename()修改

print(df)

print(df.rename(index = {'five': 'aaa'}))
print(df.rename(columns = {'pop': 'population'}))

总结

至此，一共学习三种数组：

  • Python数组（list）：e.g., [1, 2, 2, 5]. 需要用for loop去接触每一个元素，可以嵌套形成多维数组；
  • Numpy数组（ndarray）：e.g., np.array([1, 2, 2, 5]). 可以做快速的矢量化计算，是Python进行数据分析的基础，也可以嵌套形成多维数组；
  • Pandas Series：e.g., pd.Series([1, 2, 2], index = [...]). 可以由python数组或者Numpy数组作为内容生成，可以给每个值设置一个索引。可以是Pandas DataFrame的一行或者一列，矢量计算与Numpy数组一样。两个Series进行计算时，会自动先按照index对齐，再计算，没有对齐到的结果为NaN。

Pandas DataFrame: e.g., pd.DataFrame({...}, columns = [], index = [])

  • 与R的dataframe类似，是一个二维的数据结构（底层是一个二维Numpy数组），列表示feature，行表示observation；
  • 每一列取出来都是Series，index为原DataFrame的行索引；
  • 每一行取出来都是Series，index为原DataFrame的列索引；
  • 按条件筛选的方式与Numpy数组一样。

Pandas筛选数据的三种方式

  1. 用位置选取：选行df.iloc[0:2]，选列df.iloc[:, 0:2]；
  2. 用索引值选取：选行df.loc[['row1', 'row3']]，选列df[['col1', 'col2']]；
  3. 用布尔条件选取：满足条件的所有的行df.loc[df.col1 > 0]。
