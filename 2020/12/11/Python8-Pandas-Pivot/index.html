8. Python8-Pandas-Reshape
Sherry | 2020-12-11 | Python |

像Numpy Array的reshape一样，Pandas DataFrame也是可以reshape的。但由于DataFrame的行和列分别都是有明确意义的，并且各自有自己的index，所以reshape之前，需要思考reshape之后的DataFrame应该是个什么样子。所以，在进行coding之前，在纸上画出自己想要的形式是一个很有帮助的习惯。

Melt

如果我们生成下一个Dataset:

import pandas as pd
import numpy as np

columns = ['Sub' + x for x in np.array(np.arange(1, 9)).astype(str)]
df = pd.DataFrame(np.random.randint(40, 100, (40, 8)),
             columns = columns)
df['School'] = ['S_1'] * 20 + ['S_2'] * 20
df['Class'] = ['C_1'] * 5 + ['C_2'] * 5 + ['C_3'] * 5 + ['C_4'] * 5 + \
['C_1'] * 5 + ['C_2'] * 5 + ['C_3'] * 5 + ['C_4'] * 5
df['StudentID'] = np.arange(1, 41)
df = df[['School', 'Class', 'StudentID'] + columns]

如果我们想看每个学校的每个班级的各项平均成绩可以直接用：

# 在Jupyter中，如果换行之后，前一句可以被当作一句完整的语句，
# 系统会分开处理两句，所以这时需要在第一句后面加一个斜杠\来表示“这句没有结束”
df.groupby(['School', 'Class'])\
[['Sub1', 'Sub2', 'Sub3', 'Sub4', 'Sub5', 'Sub6', 'Sub7', 'Sub8']].mean()

如果我想看相对于每个Subject，不同的学校的平均成绩如何，应该要怎么办呢，这时，我们就需要将整个DataFrame换一种形式，将Sub变成一列，才能用GroupBy：

df2 = df.melt(id_vars = ['School', 'Class', 'StudentID'],
        value_vars = ['Sub1', 'Sub2', 'Sub3', 'Sub4', 'Sub5', 'Sub6', 'Sub7', 'Sub8'],
        var_name='Subject', value_name='Score')
df2

然后我们就能查看不同的学校在每个学科中的比较了。

df3 = df2.groupby(['Subject', 'School'])['Score'].agg([np.mean, np.min, np.max])
df3

行index出现了层次嵌套关系，如果想变成简单的index，可以用df3.reset_index()来去掉嵌套关系。如果想去掉列的嵌套关系，用df3.columns = df3.columns.droplevel()。

Pivot

Melt是将不同的列整合成两列（variable，value），一列是表示原来的列名，一列是原来列对应的值。由宽矩阵转化为长矩阵。
Pivot就是将长矩阵转化为宽矩阵。

继续用df3来做例子，先用df3 = df3.reset_index()去掉行index的嵌套关系，得到：

然后用pivot将长型转化为宽型：

df3_mean = df3.pivot(index='School', columns='Subject', values='mean')
df3_mean
