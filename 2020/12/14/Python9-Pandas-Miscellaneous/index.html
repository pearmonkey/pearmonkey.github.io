9. Python-Pandas-Miscellaneous
Sherry | 2020-12-14 | Python |

其实重要和常用的功能已经系统的介绍完了。这篇主要是查漏补缺一些function。

推荐大家熟悉这个cheat sheet，一般的学习工作需要的功能都能在里面找到类似的列子。所以我们根据这个cheat sheet查漏补缺还没有提及的功能和用法。

MultiIndex

一般情况下，我们会直接读入数据，而不是生成数据，所以手动传入MultiIndex的机会不多。更常见的是当我们按照多列GroupBy之后，Index是按照GroupBy中列的顺序进行分级，最左边的列名是最高级，最右边的列名是最低级。

import pandas as pd
import numpy as np
columns = ['Sub' + x for x in np.array(np.arange(1, 9)).astype(str)]
df = pd.DataFrame(np.random.randint(40, 100, (40, 8)),
             columns = columns)
df['School'] = ['S_1'] * 20 + ['S_2'] * 20
df['Class'] = ['C_1'] * 5 + ['C_2'] * 5 + ['C_3'] * 5 + ['C_4'] * 5 + \
['C_1'] * 5 + ['C_2'] * 5 + ['C_3'] * 5 + ['C_4'] * 5
df['StudentID'] = np.arange(1, 41)
df = df[['School', 'Class', 'StudentID'] + columns[0:3]]
df.groupby(['School', 'Class'])\
[columns[0:3]].mean()
去除嵌套关系可以用```df.reset_index()```。

query

选择符合条件的行

df.query('Sub1 > 90')
df.query('Sub1 > 90 and Sub2 > 90')

Date

日期是数据中常见的一列，Date类型不属于数值型也不属于字符型。
将Integer转化为Date：

pd.to_datetime('20200101')
pd.to_datetime('2020-01-01')
pd.to_datetime('2020/01/01')
pd.to_datetime('20200101').year
pd.to_datetime('20200101').month
pd.to_datetime('20200101').day
pd.to_datetime('202001', format = '%Y%m')

# 移动日期
import datetime
pd.to_datetime('202001', format = '%Y%m') + datetime.timedelta(days=5)

# date to string
str(pd.to_datetime('202001', format = '%Y%m') + datetime.timedelta(days=5))

str.contains

筛选符合条件的字符串

df2 = pd.DataFrame({'name': ['Bob_James', 'Hunter_James', 'Alice_Peter'],
                   'score': [90, 80, 100]})
df2
df2.name.str.contains('James')
df2[df2.name.str.contains('James')]

qcut

把Series切成分量相等（几乎相等）的小段，如果unique值的个数比分数小，则报错无法切。

pd.qcut(np.arange(10), 3)
pd.qcut(np.arange(10), 3, labels = ['low', 'medium', 'high'])
pd.qcut([1] * 10, 3, labels = ['low', 'medium', 'high']) # Error

shift & rank

纵向移动列

df['Sub1_next'] = df.Sub1.shift(-1)
df['Sub1_pre'] = df.Sub1.shift(1)
df

排序（GroupBy排序）

df['rank_overall'] = df['Sub1'].rank(ascending = False).astype(int)
df['rank_school'] = df.groupby('School')['Sub1'].rank(ascending = False).astype(int)
df['rank_class'] = df.groupby(['School', 'Class'])['Sub1']\
      .rank(ascending = False).astype(int)

df

isin

查看一个Series的每个值是否在另一个Series里面，返回的布尔数组可以当作index用于筛选行。

# 选出df1中符合条件的行：df1的x在df2的y中出现过
df1[df1.x.isin(df2.y)]

rolling

求rolling window的统计量

# 包括自己的前五个求mean
df.Sub1.rolling(5).mean()

drop_duplicates

df2 = pd.DataFrame({'name': ['Bob_James', 'Hunter_James', 'Bob_James'],
                   'score': [90, 80, 90]})
df2
df2.drop_duplicates() # 默认keep第一个
df2.drop_duplicates(keep = 'last') # 保留最后一个
df2.drop_duplicates(keep = False) # 只要重复，都去掉

Plotting

更多例子参考：https://pandas.pydata.org/pandas-docs/stable/user_guide/visualization.html

ts = pd.Series(np.random.randn(1000),
            index=pd.date_range('1/1/2000', periods=1000))
ts.cumsum().plot()

df = pd.DataFrame(np.random.randn(1000, 4),
                  index=ts.index,
                  columns=['A', 'B', 'C', 'D'])
df.cumsum().plot()
df.iloc[5].plot(kind='bar')

df2 = pd.DataFrame(np.random.rand(10, 4), columns=['a', 'b', 'c', 'd'])
df2.plot.bar()

df2.plot.bar(stacked=True)

df2.plot.barh(stacked=True)

# save plot
df2.plot.barh(stacked=True).figure.savefig('demo-file.pdf')
