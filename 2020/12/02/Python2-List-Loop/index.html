2. Python-基础知识的进阶
Sherry | 2020-12-02 | Python |

如果用Python作为数据处理和分析的工具，很多功能就不用立马掌握了，例如：面向对象编程，进程和线程。这些功能对于一个Data Scientist可以说是几乎用不到（虽然很多网上的教程都会讲这些，但是就个人经验而言，这些知识并不是必须的，如果遇到可以再学）。

做数据处理与分析，其实就是对Python变量进行操作，所以，我们要进一步掌握基础变量，以及变量之间的转化。这里，我们继续学习上一篇中提到的各种概念，并加以运用。

变量赋值

Python是可以同时给多个变量赋值的。
给不同变量赋同一个值：

a = b = c = 1
a = b = c = [1, 2, 3]

或者给不同变量不同的值：

name, score, age = "Tom", 95, 21

字符串String

插入一个知识点，python表示范围的形式是[start, end, step]，从start取到end-1，步长为step，default step为1

s = 'abcdef'

# 截取string：
print(s[1:4])
print(s[1:]) # 冒号后面不写表示直到结束
print(s[1::2]) # [start : end : step]，先截取string，再按步长取字符

# 重复string
print(s * 2)

# 粘贴string
print(s + ' yeah!')

数组List

# 截取子数组
a = [1, 6, 2, 4, 9, 6, 7]
print(a[::3]) # [start : end : step]，先或者子数组，再按步长取元素

# 重复数组
print(a * 2)

# 粘贴数组
b = ['aaa', 'bbb', 'ccc']
print(a + b)

# 检查元素是否在list里
print(4 in a)

字典Dict

d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}

# 获得所有key
print(d.keys())

# 获得所有value
print(d.values())

# nested dict
d = {'Michael': {'score': 95, 'age': 27}, 'Bob': {'score': 85, 'age': 25}}
print(d.keys())
print(d.values())
print(d['Michael'].keys())
print(d['Michael'].values())

数据转换

# Integer to String
print(str(1000))

# String to Integer
print(int("-100"))

# float number to Integer
# 向下取整
print(int(8.88))

一行for loop

如何对list中每一个元素进行相同的操作。

# 把每一个数字都除以2
a = [10, 12, 5, 7]
b = [x / 2 for x in a]
print(b)

# 把"new_"添加到每个元素的前面
a = ["york", "mexico", "city"]
b = ["new_" + x for x in a]
print(b)

# 对每个元素进行奇偶判断
a = [10, 12, 5, 7]
b = [x for x in a if x % 2 == 0]
print(b)

## 循环语句
break和continue的使用
```python
# 用break提前终止
for i in range(1, 100):
  if i > 10:
    break # 程序在此终止
  print(i)

# 用continue跳过不需要的
for i in range(1, 20):
  if i % 3 == 0:
    continue # 直接跳到下个循环，下面的代码不执行了
  print(i)

print函数

print多个变量，用逗号隔开，有两个可调参数，sep和end。seq是说明多个变量用什么隔开，default是一个空格，end是说明最后以什么结尾，default是换行(\n)

a = [1, 2, 3]
b = "hahaha"
print(a, b)
print("first print", end = '++++++++')
print(a, b, "hello world!", sep = '\t')
