994. Rotting Oranges
2020-02-03 | Leetcode | Leetcode

题目链接 https://leetcode.com/problems/rotting-oranges/

题目难度 Easy 47.0%

题目描述

In a given grid, each cell can have one of three values:

  • the value 0 representing an empty cell;
  • the value 1 representing a fresh orange;
  • the value 2 representing a rotten orange.

Every minute, any fresh orange that is adjacent (4-directionally) to a rotten orange becomes rotten.
Return the minimum number of minutes that must elapse until no cell has a fresh orange. If this is impossible, return -1 instead.
Example 1:

Input: [[2,1,1],[1,1,0],[0,1,1]]
Output: 4

题目分析

这是一道很有意思的题。坏了的橘子会把周围好的橘子也弄坏，然后慢慢传播开来。题目给定一个矩阵，然后问多少回合能将所有橘子都传染坏。如果不能就返回-1。这道题读下来就感觉比一般easy的题要难一点。这里的一回合，意思是所有为2的点都开始传播，这里就应该要想到BFS了，也就是tree的level traversal。level traversal就是用queue去储存一个level的点，然后遍历这个queue，做好标记，将新来的点加到后面，一直到queue为空。
在遍历的过程中，我们需要做的事情有：

  1. 对于每一个点都要看看四周是否有坏橘子；
  2. 将新的点加入queue并且标记2；
  3. 随时记录为1的点，因为最后还要判断是否所有橘子都坏了；
  4. 这里有一点是需要记住并且练习的，就是对每一个点检查上下左右四周的操作。

完整代码

class Solution {
    public int orangesRotting(int[][] grid) {
        int n = grid.length;
        int m = grid[0].length;
        Queue<int[]> q = new LinkedList();
        int freshcount = 0;
        for(int i = 0; i < n; i++){
            for(int j = 0; j < m; j++){
                if(grid[i][j] == 2)
                    q.offer(new int[]{i, j});
                else if(grid[i][j] == 1)
                    freshcount++;
            }
        }
        if(freshcount == 0) return 0;
        int count = 0;
        int[][] steps = <!--￼1-->;
        while(!q.isEmpty()){
            count++;
            int size = q.size();
            for(int i = 0; i < size; i++){
                int[] pos = q.poll();
                for(int[] step : steps){
                    int x = pos[0] + step[0];
                    int y = pos[1] + step[1];
                    if(x < 0 || y < 0 || x >= n || y >= m || grid[x][y] == 2 || grid[x][y] == 0)
                        continue;
                    grid[x][y] = 2;
                    q.offer(new int[]{x, y});
                    freshcount--;
                }
            }
        }
        return freshcount == 0 ? count - 1 : -1;
    }
}
