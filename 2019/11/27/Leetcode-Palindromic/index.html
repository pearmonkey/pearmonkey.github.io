回文字符串系列
2019-11-27 | Leetcode | Leetcode 字符串

问题描述

回文字符串英文叫 palindromic string/substring。所谓“回文”，就是正着读和反着读都是相同的字符序列，例如“level”，“noon”。所有回文字符串可以分为两大类：1）奇数回文，如“level”，中间有一个单独字符，两边延展的字符都相同；2）偶数回文，如“noon”，没有中间单独字符，对折的字符都相同。所以，在遇到回文字符串的题，这两种都要考虑进来。

首先，先不看题，我们自己能想到的问题有哪些。

  1. 最基本的是判断一个字符串是否是回文串。完成这个功能有两种方式：
    a. 从最左和最右开始往里检查
public static boolean isPalindromic(String s){
  int l = 0, r = s.length() - 1;
  while(l < r){
      if(s.charAt(l) != s.charAt(r))
          return false;
      l++;
      r--;
  }
  return true;
}

b. 从中间开始，往两边检查

	public static boolean isPalindromic(String s){
      int len = s.length();
      int l = (len - 1) / 2;
      //如果长度是奇数，r与l相等；是偶数，则l+1
    int r = len % 2 == 0 ? l + 1 : l;
      while(l >= 0 && r < len){
          if(s.charAt(l) != s.charAt(r))
              return false;
          l--;
          r++;
      }
      return true;
  }
  1. 一些字符能否组成一个回文串。这个判断只需要count每个字符的出现次数，考虑的可能性还是偶回文和奇回文：
public static boolean CanBePalindromic(String s){
    //因为不确定是否只有字母，所以假设s可能包含任何ASCII字符
    int[] count = new int[128];
    for(char c : s.toCharArray())
        //既然考虑到了所有ASCII字符，就可以直接将char强转为int
        count[(int)c]++;
    boolean odd_flag = false;
    for(int n : count){
        //如果之前已经出现过奇数字符，现在又出现了，就直接返回false
        if(n % 2 == 1 && odd_flag == true)
            return false;
        //上面没有返回false说明目前有可能还没有出现奇数字符
        if(n % 2 == 1)
            odd_flag = true;
    }
    return true;
  }
