Leetcode-必备小知识点（持续更新）
2019-12-23 | Leetcode | Leetcode

刷题的时候有一些小的知识点很有用也很快捷，但感觉很多也很杂。所以在这篇文章中持续更新总结刷题过程中所遇到的。

String / StringBuilder

序号 功能 代码
1 String start with str.startsWith(String prefix, int toffset);
2 String split str.split("\\s+"); str.split("\\.");
3 String trim str.trim();
4 Equal String str1.equals(str2);
5 String replace str.replace("", "");
6 StringBuilder sb.append(str); sb.toString(); sb.reverse();
7 Check char is letter Character.isLetter(c);
8 Find substring index str.indexOf(s, start_pos);
9 toLowerCase / toUpperCase s.toLowerCase(); s.toUpperCase();
10 strip punctuation and split s.replaceAll("[^a-zA-Z ]", " ").toLowerCase().split("\\s+");
11 StringBuilder delete char sb.deleteCharAt(i);
12 String Compare s1.compareTo(s2) < 0; //check if s1 < s2 lexicographically

List

序号 功能 代码
1 Reverse Collections.reverse(list);
2 Equals list1.equals(list2);
3 Remove last element list.remove(list.size() - 1);
4 Add as first element list.add(0, n);
5 Set element list.set(2, n);
6 Map value list to nested list new ArrayList(map.values());

序号 功能 代码

3 Get peek peek();

6 Empty q.empty();
7 clone new LinkedList<>(myQueue);

7 Initialize Deque<String> q = new ArrayDeque();

PriorityQueue

序号 功能 代码

5 ListNode Comparator new PriorityQueue<>((n1, n2)->n1.val - n2.val);

Array

序号 功能 代码

3 数组复制 b = arr.clone();

序号 功能 代码
